# 高校清单

## 一、总览

### 说明

该清单的高校数据采集自**教育部的2018全国普通高等学校名单**，根据各个学校的办学层次、往年录取分数线、2018年招生计划(分专业)方面数据，综合筛选出的清单。

所筛选的高校全部位于北京、天津、河北。对此的考虑是。。。
