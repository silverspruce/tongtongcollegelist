### 说明

- **数据来源**


- **筛选规则**

```word
1、高校的范围是河北省。
2、高校的办学层次是否包含专科
3、2018年招生计划(详细专业)
4、2017年录取分数线
```

- **学校推荐**

```word
1、此清单是不分专业，只筛选出分数可能达到学校录取最低分数线的学校清单
1、选择学校的方式可以是根据想报考专业和往年录取分数线自高到低进行选择
2、由于部分学校未公开往年录取分数线，因此需要去咨询学校招生办
```
